# 모달 잘림 해소 및 전체 노출 계획

## 목표
- 편집/미리보기에서 모달이 길어져도 잘리지 않고 전체 구조가 노출되도록 한다.
- 스크롤은 상위 편집/미리보기 영역에서 처리되도록 정리한다.

## 작업 단계
1. 모달 중립화 CSS에서 모달 자체 `max-height`/`overflow` 제한 제거
2. 고정 레이어(fixed/inset)와 모달 박스 레이어의 크기 규칙 분리
3. iframe/일반 렌더 CSS 동기화
4. 타입 검사 및 테스트 실행
5. 변경 사항 커밋

## 확인 항목
- 긴 모달이 잘리지 않고 아래쪽까지 보이는지
- 편집 화면/미리보기 화면 모두 동일하게 동작하는지
