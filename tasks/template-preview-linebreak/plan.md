# 템플릿 편집/미리보기 줄바꿈 동기화 계획

1. 현상 재현: 템플릿 수정 페이지에서 편집기와 미리보기 컴포넌트 구조 및 데이터 흐름을 확인한다.
2. 원인 분석: 편집기 값이 미리보기로 전달되는 과정에서 줄바꿈 처리 방식 차이를 추적한다.
3. 수정 설계: 공통 포맷터 적용 또는 미리보기 렌더링 방식을 조정해 Enter 입력 유무가 동일하게 반영되도록 정의한다.
4. 구현: 코드 수정 및 필요한 유틸/스타일 보완을 수행한다.
5. 검증: 해당 화면을 위한 단위/스냅샷 로직이 있다면 업데이트하고 수동으로 줄바꿈 동작을 확인한다.

## 추가 요구사항 (2024-??)
- 미리보기에서 편집기 내용이 두 번 반복되는 현상을 제거하고, 메일 수신 시와 동일한 HTML을 보여주도록 렌더링을 단순화한다.
- 메일 본문 미리보기 시 드래그하면 보이는 숨겨진 텍스트(배경과 동일한 색상)를 제거할 수 있도록 렌더링 전 전처리를 추가한다.
