# SMTP 페이지 빌드 오류 조사 및 수정 계획

1. 재현 로그를 검토하고 sqlite3 관련 JSON 파싱 오류와 Next.js 타입 오류의 원인을 파악한다.
2. 문제 있는 구성/코드를 찾아 수정하고 필요 시 의존성을 정리한다.
3. 수정 후 관련 빌드 또는 타입 검증을 수행해 오류가 해결됐는지 확인한다.
4. 빌드 단계에서 `DATABASE_URL` 강제 조건 때문에 실패하는 케이스를 재현하고 안전한 SQLite 대체 전략을 적용한다.
5. 런타임 콘솔에 보고된 `highcontrast.js` 구문 오류와 React 오류 #185의 원인을 파악하고 해결책을 제시한다.
