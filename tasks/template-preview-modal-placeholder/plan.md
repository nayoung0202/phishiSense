# 템플릿 미리보기 모달 플레이스홀더 전역 적용 계획

## 목표
- 템플릿 생성/수정 미리보기에서 HTML 모달이 실제 팝업처럼 뜨지 않도록 전역 처리한다.
- 현재 템플릿뿐 아니라 이후 생성되는 템플릿에도 동일 규칙이 자동 적용되게 한다.

## 작업 단계
1. 미리보기 렌더링 경로(생성/수정 공통 컴포넌트) 식별
2. 미리보기 전용 HTML 후처리 유틸 구현
   - 모달 관련 노드를 플레이스홀더로 치환
   - 모달 자동 실행 스크립트/속성 비활성화
3. 미리보기 컴포넌트에 후처리 유틸 적용
4. 영향 범위 점검 및 타입 검사 실행
5. 결과 정리 및 커밋

## 확인 항목
- 미리보기에서 모달 오버레이가 뜨지 않는지
- 에디터 내 삭제/선택 조작이 정상 동작하는지
- 생성 화면과 수정 화면에서 동일하게 적용되는지
