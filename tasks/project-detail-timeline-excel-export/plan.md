# 프로젝트 상세 이벤트 타임라인 엑셀 내보내기 기능 계획

## 목표
- 프로젝트 상세 화면의 이벤트 타임라인 데이터를 엑셀 파일(.xlsx)로 다운로드할 수 있도록 한다.

## 작업 단계
1. 타임라인 데이터 경로 확인
   - 프로젝트 상세 화면에서 이벤트 타임라인이 사용하는 API와 데이터 스키마를 확인한다.
2. 서버 다운로드 API 추가
   - 프로젝트 ID 기준으로 타임라인 데이터를 조회해 엑셀 바이너리 응답을 반환하는 엔드포인트를 구현한다.
3. 프론트엔드 버튼 추가
   - 프로젝트 상세 이벤트 타임라인 영역에 "엑셀 다운로드" 액션을 추가하고 API 호출 후 파일 다운로드를 수행한다.
4. 검증
   - 타입 체크와 관련 테스트(또는 최소 스모크 검증)를 수행한다.
5. 커밋

## 확인 항목
- 다운로드 파일에 타임라인 주요 컬럼(시간/유형/대상/내용 등)이 포함되는지
- 데이터가 없을 때도 동작이 깨지지 않는지
- 한글 파일명/콘텐츠가 정상 인코딩으로 내려가는지
