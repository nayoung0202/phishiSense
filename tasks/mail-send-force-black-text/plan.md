# 메일 발송 시 텍스트 색상 검정 보정 계획

## 목표
- 메일 발송 직전에 HTML 텍스트 색상을 점검해 white 계열(`white`, `#fff`, `#ffffff`)을 검정 계열(`#111111`)으로 보정한다.
- 에디터 저장본은 변경하지 않고 발송 파이프라인에서만 적용한다.

## 작업 단계
1. 공통 유틸 구현
   - 발송용 HTML 문자열에서 텍스트 색상 선언을 검사/치환하는 유틸을 추가한다.
   - 인라인 `style`, `<style>` 블록, `color` 속성 케이스를 처리한다.
2. 발송 경로 통합
   - `sendJobRunner`, `send-worker`, `test-send route`, `server/mailer`의 발송 직전에 공통 유틸을 적용한다.
3. 테스트 추가 및 검증
   - 유틸 단위 테스트를 추가한다.
   - `npm run check`와 신규 테스트를 실행한다.
4. 변경사항 커밋

## 확인 항목
- 발송 직전 HTML에서 white 텍스트 선언이 검정으로 치환되는지
- 기존 발송 플로우(제목/헤더/치환 링크)가 깨지지 않는지
