# 프로젝트 상세 부서별 분포 발송 수 기준 변경

## 목표
프로젝트 상세 화면의 `부서별 분포` 차트/툴팁 값을 발송 수(대상자 수) 기준으로 일치시킨다.

## 작업 범위
- 현재 부서별 분포 집계 로직 확인
- 발송 수 기준으로 집계 기준 정의
- 차트 데이터 생성 로직 수정
- 타입 체크로 회귀 확인

## 수행 단계
1. 액션 로그/대상자 데이터에서 발송 수 계산 가능한 필드를 확인한다.
2. 부서별 분포 데이터를 발송 수 기반으로 재계산하도록 변경한다.
3. 데이터가 없을 때의 표시를 왜곡 없이 처리한다.
4. 타입 체크를 실행해 변경 안정성을 확인한다.
