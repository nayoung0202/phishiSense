# 모달 미리보기 스크롤 및 배경 동기화 계획

## 목표
- 템플릿 편집/미리보기에서 모달 내용이 길면 내부 스크롤로 전체 확인 가능하게 한다.
- 모달 뒤 레이어는 어두운 오버레이 대신 템플릿 배경이 그대로 보이게 한다.

## 작업 단계
1. 모달 중립화 CSS 규칙 보강
   - 모달 영역 최대 높이와 `overflow-y: auto` 적용
   - 고정 레이어(fixed/inset) 배경 투명화
2. iframe 미리보기 CSS 상수(`src/lib/templatePreview.ts`) 수정
3. 일반 미리보기 CSS(`src/app/globals.css`) 동일 반영
4. 타입 검사 및 테스트 실행
5. 변경 커밋

## 확인 항목
- 긴 모달에서 스크롤로 하단 내용까지 확인 가능한지
- 모달 뒤 배경이 템플릿 원래 배경처럼 보이는지
- 편집/미리보기 모두 동일하게 동작하는지
