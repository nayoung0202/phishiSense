# 템플릿 편집기 모달 비활성 렌더링 적용 계획

## 목표
- 메일 본문/악성 메일 본문 편집 모드에서 모달 HTML이 실제 팝업처럼 뜨지 않도록 한다.
- 미리보기처럼 모달 구조는 보이되, 상호작용 없이 내용만 확인되도록 한다.

## 작업 단계
1. 편집 모드 렌더링 컴포넌트 확인
2. 편집 영역에 모달 비활성 샌드박스 클래스 적용
3. 타입 검사 및 관련 테스트 실행
4. 변경 사항 커밋

## 확인 항목
- 수정 화면 진입 시 모달 오버레이가 화면을 덮지 않는지
- 편집기에서 모달 HTML 내용은 보이는지
- 기존 편집/저장 흐름이 유지되는지
