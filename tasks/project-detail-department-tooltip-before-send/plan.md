# 프로젝트 상세 부서별 분포 발송 전/후 표시 정리

## 목표
부서별 분포는 발송 후에만 원형 차트/툴팁을 표시하고, 발송 전에는 툴팁이 뜨지 않도록 차트를 숨긴다.

## 작업 범위
- 실제 발송 여부 판별 가능한 필드 확인
- 상세 화면의 부서별 분포 표시 조건 수정
- 발송 후 툴팁은 부서별 발송 수(n명) 노출
- 발송 전 안내 UI 정리
- 타입 체크 및 커밋

## 수행 단계
1. 액션 로그 API와 발송 처리 코드에서 `sentAt`/발송 상태를 점검한다.
2. 필요 시 액션 로그 API 응답에 발송 시각 필드를 추가한다.
3. 상세 화면에서 발송 전/후 분기 렌더링을 적용한다.
4. 타입 체크 후 커밋한다.
