# SMTP 관리 페이지 구축 계획

## 목표
관리자 전용 SMTP 관리 페이지와 관련 API 연동, UI 컴포넌트, 타입 정의를 추가하여 SMTP 설정과 테스트 발송을 프론트엔드에서 제어할 수 있게 한다.

## 작업 항목
1. 프로젝트 구조 파악 및 기존 라우팅/관리자 메뉴 위치 확인.
2. SMTP 관련 타입 정의와 API 래퍼 함수 추가 (`client/src/types`, `client/src/lib`).
3. 테넌트 선택, SMTP 설정 폼, 테스트 패널 컴포넌트를 설계 및 구현.
4. `/admin/smtp` 페이지 작성 후 React Router(or equivalent)에 라우트 연결, 관리자 메뉴 링크 추가.
5. 폼 검증 로직, UX 요구사항(자동 port-security 동기화, 비밀번호 처리, 토스트/상태 갱신 등) 구현.
6. GET/PUT/POST API 연동, 요청/응답 상태에 따라 UI 업데이트 및 결과 표시.
7. 신규 기능에 대한 기본 테스트(가능한 범위) 또는 수동 점검 시나리오 기록.
8. 변경사항 검토, 필요 시 린트/타입 검사 실행 후 정리.
