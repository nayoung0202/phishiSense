# 작업 계획

## 목표
- Next.js 빌드 실패 원인(`pg` 타입 선언 누락)을 해결한다.

## 범위
- 타입 선언 추가 또는 선언 파일 보강
- 필요한 경우 빌드 스크립트 재확인

## 단계
1. `pg` 타입 선언 제공 방식 확인(패키지 설치 여부/대체 선언 파일 필요 여부)
2. 최소 변경으로 타입 오류 제거
3. 빌드 재실행으로 확인

## 리스크 및 대응
- 선언 파일 추가 시 모듈 해석 충돌 가능성 → `tsconfig`/`types` 범위 점검

## 완료 기준
- `npm run build`에서 `pg` 관련 타입 오류가 재현되지 않는다.
