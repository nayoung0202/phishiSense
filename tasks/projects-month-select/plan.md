# 프로젝트 페이지 월별 Select 추가 계획

## 목표
대시보드와 동일하게 프로젝트 관리 페이지에서도 월을 선택해 해당 월의 프로젝트만 볼 수 있도록 Select 필터를 추가한다.

## 작업 단계
1. **요구 분석 및 데이터 파이프라인 점검**: `Projects.tsx`에서 현재 연도/분기 필터, 데이터 구조, 캘린더 연동 방식을 파악한다.
2. **월 옵션/상태 로직 구현**: 분기 프로젝트 목록에서 월별 옵션을 생성하고 상태를 추가한 뒤 선택된 월에 맞게 데이터를 필터링한다.
3. **UI 통합 및 동작 검증**: Select 컴포넌트를 상단 필터 영역에 추가하고, 리스트/보드/캘린더 등에서 필터링 결과가 일관되게 반영되는지 확인한다.
