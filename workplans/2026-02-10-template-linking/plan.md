# 작업 계획

- 목표: 메일 본문/악성 본문/훈련 안내 페이지 연결 규칙을 반영하고, 자동 링크 삽입/검증/경고 UI까지 적용

## 단계
1. 템플릿/프로젝트 스키마 확장(자동 삽입 설정, 검증 오류 저장) 및 기본값 반영
2. 토큰 추출/검증/메일 본문 생성 로직 추가 및 워커/발송 API에 적용
3. 템플릿 편집 UI에 상태 배지/자동 삽입 설정 패널 추가
4. 미리보기에서 자동 삽입 결과 표시 및 "링크 삽입" 버튼 추가
5. SUBMIT_URL 토큰 추가 및 /p 제출 플로우/보안 헤더 적용
6. 예약/수동 시작 시 검증 실패 처리와 오류 표시 반영
