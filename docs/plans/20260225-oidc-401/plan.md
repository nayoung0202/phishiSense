# OIDC 콜백 401 원인 분석 계획

## 목표
- `/oidc/callback` 요청에서 401이 발생하는 원인을 코드와 설정에서 확인하고, 재현 및 수정 방향을 제시한다.

## 작업 범위
- OIDC 관련 라우팅/미들웨어/쿠키/세션 처리 확인
- 환경 변수(.env)와 콜백 검증 로직 확인
- 401 응답 생성 지점 추적

## 할 일
1. 서버/클라이언트에서 `oidc` 관련 라우트/핸들러/미들웨어를 검색한다.
2. 401 응답을 반환하는 코드 경로를 찾아 조건을 정리한다.
3. `.env` 또는 설정에서 필요한 값(issuer, client_id, redirect_uri 등)과 실제 요청 값을 비교한다.
4. 원인 가설과 수정/검증 체크리스트를 정리한다.
