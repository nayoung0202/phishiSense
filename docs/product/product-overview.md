# 제품 개요

## 목적

PhishSense는 기업 내부 보안 훈련을 위해 피싱 시뮬레이션 프로젝트를 설계, 발송, 추적, 보고하는 운영 콘솔입니다.

## 주요 사용자

- 보안 운영자: 훈련 프로젝트를 생성하고 발송 상태를 관리합니다.
- 교육 담당자: 대상자, 교육 페이지, 결과 보고서를 관리합니다.
- 훈련 대상자: 메일 링크를 통해 공개 훈련/교육 페이지에 진입합니다.

## 핵심 기능

- OIDC 기반 로그인과 tenant/onboarding 분기
- tenant 준비 완료 후 빈 상태 tenant를 위한 setup checklist/empty state 진입
- 훈련 프로젝트 생성, 복제, 일정 관리, 상태 관리
- 대상자, 메일 템플릿, 교육 페이지 관리
- SMTP 설정과 발송 워커 기반 메일 전송
- 추적 링크, 오픈/클릭/제출 이벤트 집계
- 보고서 설정, 보고서 생성, 파일 다운로드
- `platform-api` 연동을 통한 tenant/entitlement 동기화

## 주요 흐름

1. 사용자가 로그인하면 제품은 세션과 플랫폼 컨텍스트를 확인합니다.
2. tenant 또는 entitlement가 준비되지 않았으면 onboarding 흐름으로 이동합니다.
3. 준비가 끝났지만 업무 데이터가 없는 tenant는 setup checklist 또는 empty state로 초기 구성을 안내받습니다.
4. 이후 프로젝트, 템플릿, 대상자, 교육 페이지를 조합해 훈련을 구성합니다.
5. 발송 워커가 메일을 전송하고, 추적 이벤트와 보고서 데이터가 누적됩니다.
6. 운영자는 결과를 대시보드와 보고서로 확인합니다.
