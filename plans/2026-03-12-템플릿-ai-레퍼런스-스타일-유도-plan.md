# 템플릿 AI 레퍼런스 스타일 유도 계획

## 목적
- AI 템플릿 생성 프롬프트에 배송 재확인형 레퍼런스 HTML 구조를 반영한다.
- 메일본문과 악성메일본문이 같은 주제와 톤을 유지하면서도, 제공된 예시와 비슷한 구조와 시각적 밀도를 갖도록 유도한다.

## 변경 범위
- `src/server/services/templateAi.ts`
- `src/server/services/templateAi.test.ts`

## 구현 기준
- 메일본문용 레퍼런스 HTML 구조를 프롬프트에 추가
- 악성메일본문용 모달/카드형 레퍼런스 HTML 구조를 프롬프트에 추가
- `{{LANDING_URL}}`, `{{TRAINING_URL}}` 플레이스홀더 규칙은 유지
- 프롬프트 빌더를 테스트 가능하게 정리

## 검증 항목
- 프롬프트에 메일본문 레퍼런스 HTML이 포함된다.
- 프롬프트에 악성메일본문 레퍼런스 HTML이 포함된다.
- 프롬프트가 레퍼런스를 그대로 복사하지 말고 구조적으로 유사하게 생성하라고 지시한다.
