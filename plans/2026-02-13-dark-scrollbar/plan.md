# 작업 계획

## 목표
- 다크 모드에서 사이트 내부 스크롤바가 전체 페이지 스크롤바와 유사한 스타일을 갖도록 맞춘다.

## 범위
- Radix ScrollArea 기반 스크롤바
- 에디터 영역 등 네이티브 스크롤바(`.editor-scrollbar`)

## 작업 단계
1. 현재 스크롤바 스타일 정의 위치와 적용 범위를 확인한다.
2. 다크 모드 전용 스크롤바 스타일을 추가해 전체 페이지 스크롤바 톤과 정렬한다.
3. 적용 대상 컴포넌트/클래스에 스타일이 반영되는지 확인한다.

## 검증
- 다크 모드에서 내부 스크롤바 색상과 둥근 형태가 전체 페이지 스크롤바와 유사하게 보인다.
