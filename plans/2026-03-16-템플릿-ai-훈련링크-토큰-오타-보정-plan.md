# 템플릿 AI 훈련링크 토큰 오타 보정 계획

## 작업 목적
- AI 생성 템플릿에서 `{{TRAINING_URL}}` 토큰이 오타 형태로 저장되어 프로젝트 생성 시 훈련안내페이지 링크 치환이 실패하는 문제를 해결한다.
- 흔한 오타(`tranning`, 중괄호 개수 오류, 소문자 등)가 들어와도 서버에서 표준 토큰으로 정규화되도록 만든다.

## 확인 범위
- `src/server/services/templateAi.ts`
- `shared/templateTokens.ts`
- 프로젝트 생성/발송 시 훈련 링크 치환에 사용하는 경로
- 관련 테스트 파일

## 작업 계획
- 훈련 링크 토큰 치환이 정확히 어떤 문자열에만 반응하는지 확인한다.
- AI 생성 후보 정제 단계에서 훈련 링크 토큰의 흔한 오타를 표준 `{{TRAINING_URL}}`로 보정한다.
- 필요하면 저장 전 검증 메시지와 테스트를 보강해 동일 문제가 다시 들어오지 않도록 한다.

## 검증 계획
- `{{tranning_url}}}`, `{{training_url}}`, 여분 괄호가 포함된 형태가 표준 토큰으로 정규화되는지 확인한다.
- 프로젝트 생성/미리보기 경로에서 정규화된 토큰이 실제 훈련 링크로 치환되는 전제 조건이 유지되는지 테스트한다.
