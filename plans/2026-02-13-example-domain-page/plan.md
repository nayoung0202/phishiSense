# 작업 계획

## 목표
- 예시 도메인 페이지를 한국어로 구성하고 PhishSense 톤에 맞춘 디자인을 적용한다.
- 템플릿/훈련 안내 미리보기 링크가 새 예시 페이지를 사용하도록 맞춘다.

## 범위
- 예시 도메인 HTML 응답 라우트
- 미리보기용 URL 상수

## 작업 단계
1. 예시 도메인 전용 라우트를 추가하고 한국어 안내 문구/스타일을 작성한다.
2. 미리보기 URL을 새 예시 페이지로 변경한다.
3. 미리보기 링크 동작과 스타일을 확인한다.

## 검증
- 예시 페이지가 한국어로 표시되고 PhishSense 색상 계열이 적용된다.
- 미리보기 링크가 예시 페이지로 연결된다.
