# 프로젝트 관리 보드 상태 컬럼 가시성 및 레이아웃 조정 계획

## 작업 목적
- 보드 뷰에서 프로젝트가 없는 상태 컬럼은 숨긴다.
- 표시 컬럼이 4개일 때는 한 줄 4칸 대신 2x2 그리드로 배치해 균형을 맞춘다.

## 수정 범위
- `src/features/Projects.tsx`
- `src/features/Projects.test.tsx`

## 검증 항목
- 프로젝트가 있는 상태 컬럼만 노출되는지 확인
- 상태 컬럼 순서가 예약, 진행중, 완료, 임시로 유지되는지 확인
- 표시 컬럼 수가 4개일 때 2열 레이아웃 클래스를 사용하는지 확인
