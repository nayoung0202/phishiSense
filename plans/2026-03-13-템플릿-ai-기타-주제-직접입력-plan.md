# 템플릿 AI 기타 주제 직접 입력 계획

## 목적
- AI 템플릿 생성에서 기존 주제 목록에 없는 시나리오도 만들 수 있도록 `기타 + 주제 직접 입력` 흐름을 추가한다.

## 변경 범위
- `shared/templateAi.ts`
- `src/components/TemplateAiGenerateDialog.tsx`
- `src/components/TemplateAiGenerateDialog.test.tsx`
- `src/server/services/templateAi.ts`
- `src/server/services/templateAi.test.ts`
- `src/app/api/templates/ai-generate/route.test.ts`

## 구현 내용
1. 공통 요청 스키마에 `topic: "other"`와 `customTopic`을 추가한다.
2. `topic`이 `other`일 때 `customTopic` 입력을 필수로 검증한다.
3. 생성 모달에서 `기타` 선택 시 `주제 직접 입력` 필드를 노출한다.
4. 예상 크레딧 계산과 생성 요청에 `customTopic`을 반영한다.
5. 서버 프롬프트에서 실제 생성 주제를 `customTopic` 또는 선택 주제 라벨 기준으로 사용한다.
6. UI/스키마/프롬프트 관련 테스트를 보강한다.

## 검증 기준
- `기타` 선택 시 직접 입력 없이 생성할 수 없다.
- 직접 입력한 주제가 API 요청에 포함된다.
- 서버 프롬프트가 직접 입력 주제를 반영한다.
- 기존 고정 주제 선택 흐름은 그대로 동작한다.
