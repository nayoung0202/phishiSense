# 템플릿 AI 생성 문구 한글화 계획

## 작업 목적
- 템플릿 AI 생성 1차 기능에 추가된 영문 UI 문구와 오류 메시지를 한국 출시 기준에 맞게 한글로 정리한다.

## 수정 범위
- `shared/templateAi.ts`
- `src/components/TemplateAiGenerateDialog.tsx`
- `src/server/services/templateAi.ts`
- `src/app/api/templates/ai-generate/route.ts`
- 관련 테스트

## 검증 항목
- 사용자에게 노출되는 선택값, 버튼, 안내 문구가 한글인지 확인
- 서버 검증 오류와 API 오류 문구가 한글인지 확인
- 테스트 기대값이 변경 문구와 일치하는지 확인
