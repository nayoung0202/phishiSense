# 작업 계획

## 목표
- 링크 삽입 다이얼로그의 주요 문구를 한글로 표시한다.

## 범위
- SunEditor 언어 설정 및 링크 다이얼로그 문구

## 작업 단계
1. SunEditor 언어 팩 적용 여부를 확인한다.
2. 링크 다이얼로그 문구를 한글로 오버라이드한다.
3. 실제 링크 삽입 다이얼로그에서 문구가 변경되었는지 확인한다.

## 검증
- Insert Link, URL to link, Text to display, Open in new window, Download Link, Submit 문구가 한글로 표시된다.
