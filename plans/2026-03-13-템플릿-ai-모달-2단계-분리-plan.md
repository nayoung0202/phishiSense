# AI 템플릿 생성 모달 2단계 분리 계획

## 목적
- AI 템플릿 생성 모달을 `옵션 입력`과 `후보 비교`의 2단계 흐름으로 분리한다.
- 모달 재오픈 시 이전 상태가 남지 않도록 항상 초기화한다.

## 변경 범위
- `src/components/TemplateAiGenerateDialog.tsx`
- `src/components/TemplateAiGenerateDialog.test.tsx`

## 구현 기준
- 같은 모달 안에서 `options` / `candidates` step 상태를 전환한다.
- 1단계는 옵션 입력, 예상 크레딧, 경고 문구, 생성 버튼만 보여준다.
- 2단계는 후보 비교, 전체 재생성, 선택 제외 재생성, 후보 반영, 옵션 다시 설정 버튼만 보여준다.
- 1단계로 돌아갈 때 입력값은 유지하고, 모달을 닫으면 모든 상태를 초기화한다.
- 기존 API 계약과 세션 초안 반영 흐름은 유지한다.

## 검증 항목
- 최초 오픈 시 1단계만 보인다.
- 후보 생성 성공 후 2단계로 전환된다.
- 2단계에서 1단계로 돌아가도 입력값이 유지된다.
- 모달을 닫았다가 다시 열면 완전히 초기화된다.
- 전체 재생성, 선택 제외 재생성, 후보 반영이 기존대로 동작한다.
