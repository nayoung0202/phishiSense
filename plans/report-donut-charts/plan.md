# 작업 계획

## 목표
- 보고서 생성 시 메일 열람/미열람, 링크 클릭/미클릭을 도넛 차트로 생성해 DOCX에 삽입한다.

## 범위
- 보고서 렌더러(Python) 차트 생성 로직
- 서버 매핑 레이어(차트 데이터 키/값)
- 템플릿에서 사용할 차트 키 정의

## 작업 단계
1. Python 렌더러에 도넛 차트 생성 및 InlineImage 삽입 기능을 추가한다.
2. 서버 매핑 레이어에서 차트 데이터와 키(open_donut_chart, click_donut_chart)를 구성한다.
3. 템플릿에서 사용할 플레이스홀더 키 목록을 정리한다.

## 검증
- DOCX에 도넛 차트가 크기(13.5cm x 5cm)로 삽입된다.
- 열람/미열람, 클릭/미클릭 값이 정확히 반영된다.
