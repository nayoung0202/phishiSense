# 작업 계획

## 목표
- 템플릿 관리의 미리보기 UI를 훈련 안내 페이지 미리보기와 동일한 형태로 맞춘다.

## 범위
- 템플릿 관리 화면(템플릿 리스트/카드/미리보기 모달)
- 훈련 안내 페이지 미리보기 구현부

## 작업 단계
1. 템플릿 관리와 훈련 안내 페이지의 미리보기 렌더링 위치/구현을 파악한다.
2. 훈련 안내 페이지 미리보기 구성 요소를 재사용하도록 템플릿 미리보기 UI를 수정한다.
3. 스타일과 동작(스크롤, 프레임, 버튼)을 정렬하고 불일치 요소를 정리한다.

## 검증
- 템플릿 관리의 미리보기와 훈련 안내 페이지 미리보기 UI/동작이 동일하게 보인다.
