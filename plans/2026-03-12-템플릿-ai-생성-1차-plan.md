# 템플릿 AI 생성 1차 구현 계획

## 작업 목적
- 템플릿 관리 페이지의 기존 `AI 템플릿 생성` 버튼을 활성화한다.
- 메일본문과 악성메일본문을 한 세트로 생성하는 AI 후보 4개를 모달에서 비교하고, 선택 결과를 새 템플릿 수정 화면에 반영한다.

## 수정 범위
- `shared/templateAi.ts`
- `src/server/services/templateAi.ts`
- `src/app/api/templates/ai-generate/route.ts`
- `src/components/TemplateAiGenerateDialog.tsx`
- `src/features/Templates.tsx`
- `src/features/TemplateEdit.tsx`
- 관련 테스트 및 `.env.example`

## 검증 항목
- 후보 세트 4개 생성 및 선택 후보 제외 일괄 재생성
- 예상 크레딧 소모 표시
- 선택 후보를 새 템플릿 작성 화면에 제목 포함 전체 덮어쓰기 반영
- 외부 리소스/스크립트 차단 검증 및 필수 토큰 규칙 확인
