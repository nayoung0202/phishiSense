# 작업 계획

## 목표
- 템플릿 관리 및 훈련 안내 페이지 에디터의 폰트/사이즈 선택 드롭다운에서 글자 대비 문제를 해결한다.

## 범위
- 에디터 UI(select/드롭다운) 스타일
- 관련 전역 스타일(예: globals.css, 에디터 전용 CSS)

## 작업 단계
1. 에디터 종류와 select 렌더링 요소를 확인한다.
2. 드롭다운/옵션 텍스트 대비가 낮아지는 CSS 원인을 찾는다.
3. 에디터 전용 범위에서 텍스트/배경 색을 명시해 가독성을 확보한다.

## 검증
- 폰트/폰트 사이즈 선택 목록이 라이트/다크 모두에서 읽힌다.
