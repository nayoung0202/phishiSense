# 프로젝트 생성 화면 안전·알림 블록 제거 계획

## 목적
- 프로젝트 생성 화면에서 `안전 · 알림` 섹션을 제거해 입력 단계를 단순화한다.

## 변경 범위
1. `src/features/ProjectCreate.tsx`에서 `안전 · 알림` 카드 UI 제거
2. 제거된 UI에서만 사용하던 상태값(`notificationInput`) 정리
3. 나머지 프로젝트 생성 로직(기본값, payload 구조)은 기존 동작 유지

## 검증
- 프로젝트 생성 화면에서 `안전 · 알림` 카드가 노출되지 않는다.
- 화면 렌더링/제출 버튼 동작에 타입 오류가 없다.
