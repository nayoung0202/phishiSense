# 프로젝트 관리 월 선택에 분기 전체 옵션 추가

## 목표
- 프로젝트 관리 화면의 월 선택 드롭다운에 "분기 전체" 옵션을 추가한다.
- 선택 시 해당 분기 전체 프로젝트가 표시되도록 한다.

## 작업 범위
- `src/features/Projects.tsx`의 월 옵션 생성 및 필터링 로직 수정
- 월 선택 변경 핸들러에서 "분기 전체" 값 처리

## 절차
1. 월 옵션 생성 로직에 분기 전체 항목 추가
2. 월 선택 변경 및 기본 선택 로직 보정
3. 분기 전체 선택 시 필터링 우회 확인

## 검증
- 2025년 3분기 선택 후 "분기 전체"를 선택했을 때 분기 전체 프로젝트 표시 확인
