# 2026년 1분기 프로젝트 지표 데이터 수정 계획

## 목표
- 2026년 1월 및 2026년 1분기 프로젝트 데이터의 발송수/오픈률/클릭률/제출률을 실제 진행된 것처럼 보이도록 수정한다.
- 각 프로젝트의 지표 값이 서로 다르게 보이도록 조정한다.

## 작업 범위
- 프로젝트 데이터가 정의된 시드/스토리지/데이터 파일을 확인한다.
- 2026년 1월 및 1분기(1~3월) 관련 프로젝트 항목의 지표를 수정한다.
- 변경된 지표가 UI/리포트에서 정상적으로 표시되는지 확인한다.

## 절차
1. 데이터 위치 탐색 및 관련 파일 파악
2. 2026년 1월 및 1분기 프로젝트 항목 식별
3. 발송수/오픈률/클릭률/제출률 값 업데이트(프로젝트별로 상이하게)
4. 변경사항 검토

## 검증
- 필요 시 `npm run check`로 타입 검증
- 관련 화면에서 지표 표시 확인
