# 2026년 1월·1분기 프로젝트 지표 7개 구성 계획

## 목표
- 2026년 1월과 1분기 프로젝트 데이터를 각각 7개로 구성한다.
- 발송수/오픈률/클릭률/제출률이 프로젝트별로 서로 다르게 보이도록 조정한다.

## 작업 범위
- 시드 데이터 생성 로직(프로젝트 생성 구간) 수정
- 2026년 1월 및 1분기(1~3월) 프로젝트 수와 지표 오버라이드 적용

## 절차
1. 프로젝트 시드 위치 확인
2. 2026년 1월 및 1~3월에 대해 7개 프로젝트 오버라이드 정의
3. 생성 개수 및 지표 값 적용
4. 변경사항 검토

## 검증
- 필요 시 `npm run dev`로 재시드 확인
